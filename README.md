> Application web professionnelle de gestion des sinistres automobiles avec authentification multi-rôles.

---

## 📋 Description

Cette application permet aux compagnies d'assurance de gérer efficacement leurs sinistres automobiles. Elle offre une interface moderne, des statistiques en temps réel et une gestion complète des dossiers.

### 🎯 Fonctionnalités principales

| Fonctionnalité | Description |
|----------------|-------------|
| 🔐 **Authentification** | Inscription et connexion sécurisée avec 3 rôles |
| 👥 **Multi-rôles** | Admin, Gestionnaire, Client avec des espaces dédiés |
| 📊 **Tableau de bord** | Statistiques et graphiques en temps réel |
| 📋 **CRUD complet** | Créer, lire, modifier, supprimer des sinistres |
| 🔍 **Recherche avancée** | Filtres multi-critères (statut, date, montant) |
| 📤 **Export CSV** | Exporter les données en un clic |
| 🌓 **Mode sombre/clair** | Thème personnalisable |
| 📱 **Responsive** | Adapté à tous les écrans |
| 🎨 **Design moderne** | Animations, confettis, interface premium |
